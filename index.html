Validation documentation

Add these scripts at the bottom of the body:

<script src="validation/validation.js"></script>
<script src="validation/input-validation.js"></script>

Enable validation on your form by adding the following script after the relevant dom elements have loaded (replace "formId" with an actual form id): 

var form = document.getElementById('formId');
var validation = new Validation(form);

Validation examples

Plain example

Basic validation does not ever show error but shows success when a user has entered something. This is to provide positive feedback.

Required example

Add the attribute "required" to the inputs that are mandatory. Checks if the input is has a value (after trim).

required

Pattern example

Add a pattern attribute to match against a regular expression.

pattern="^1234AB$"

Select example

Select inputs act similar to plain inputs, if user changes or blurs the input it shows as success to provide positive feedback.

Form hints example

A required validation with default and error hints.

Uses data-validation-default="Default hint." and data-validation-error="Error hint." attributes on the input and data-validation-hint-for="input_example_hints" on the form hint.

Default hint.

No icons example

A required validation without icons.

data-validation-config='{"showIcons":false}'

No auto validation example

A required validation that does not automatically validate. In this case a button is used to validate.

data-validation-config='{"auto":false}' 

var formExampleNoAuto = document.getElementById('form_example_no_auto');
var validationNoAuto = new Validation(formExampleNoAuto);
$('#button_no_auto').on('click', function (e) {
    e.preventDefault();
    document.getElementById('input_example_no_auto').inputValidation.validateRequired();
});
